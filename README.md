# Dual-Axis Solar Tracking System

## Overview
This project implements a Dual-Axis Solar Tracking System using Arduino Uno, servo motors, and light sensors. The system automatically adjusts the solar panel orientation to maximize sunlight exposure.

## Features
- Dual-axis tracking
- Horizontal and vertical servo control
- LCD monitoring
- Sensor value display
- Simulated in Wokwi

## Components
- Arduino Uno
- 2 Servo Motors
- 4 Light Sensors (simulated using potentiometers)
- 16x2 I2C LCD
- Wokwi Simulator
